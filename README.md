Утилита фильтрации содержимого файлов.

Утилита фильтрует содержимое указанных файлов в формате .txt и записывает их отдельные файлы.

Строки -> strings.txt\
Целые числа -> integers.txt\
Числа с плавающей точкой -> floats.txt

Инструкция по запуску:

    java -jar util.jar -p {prefix-name} -o {path} -s -f -a {file-name}.txt

Обязательно указать входной файл, так же есть 5 необязательных опций

-p {prefix-name}: добавляем префикс к выводящим файлам. Прим. -p simple- -> simple-strings.txt\
-o {path}: добавляем путь сохранения выводящих файлов\
-s: выводим краткую статистику отфильтрованных файлов в консоль\
-f: выводим полную статистику отфильтрованных файлов в консоль\
-a: данные записываются в конец файла, по умолчанию перезаписываются\

Краткая статистика показывает количество выводящих файлов.\
Полная статистика показывает:

для строк - длина самой короткой и длинной строки\
для чисел - минимальная, максимальная, сумма, среднее
